Лабораторная: "Люди в черном"

Консольное приложение на C# с использованием ООП: абстрактные классы, наследование, интерфейсы, полиморфизм, статические поля, работа с файлами.

Что реализовано:

  • Абстрактный класс Инопланетянин
  • Классы-наследники: Гуманоид, Арахнид, АгентMIB
  • Интерфейсы IМаскировка и IОружие
  • Sealed классы и методы
  • Конструкторы (по умолчанию, с параметрами, копирования)
  • Статистика и константы
  • Запись/чтение в файл
// Пример кода: Агент MIB
sealed class АгентMIB : Гуманоид, IОружие
{
    public const string ПАРОЛЬ_MIB = "ЧЕРНЫЙ_КОСТЮМ";
    public static int всегоАгентов = 0;
    
    public sealed override void Замаскироваться()
    {
        Console.WriteLine($"АГЕНТ {букваАгента} В ЧЕРНОМ КОСТЮМЕ!");
    }
    
    public void Нейтрализовать(Инопланетянин цель)
    {
        цель.здоровье = 0;
    }
}

Лабораторная: "Автомобили"

Работа с классами, конструкторами, списками, исключениями и файловым вводом/выводом.

Что реализовано:

  • 5 видов конструкторов
  • Собственный класс исключений CarException
  • Обработка ошибок ввода (try-catch-throw)
  • Сохранение и загрузка из файлов
  • Поиск самого "убитого" автомобиля
  • Статический счетчик объектов
// Пример кода: обработка исключений
class CarException : Exception
{
    public CarException(int x) { ошибкаГод = x; }
    public CarException(string тип, int значение) { ... }
    
    public void obrabotka()
    {
        if (ошибкаГод != 0) {
            if (ошибкаГод < 1980)
                Console.WriteLine($"Год {ошибкаГод} меньше 1980!");
        }
    }
}
